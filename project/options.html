How to get your API Key

1

Visit Google AI Studio

Go to Google AI Studio and sign in with your Google account.

2

Create API Key

Click "Create API Key" and select your Google Cloud project or create a new one.

3

Copy and Paste

Copy the generated API key and paste it into the field above.
